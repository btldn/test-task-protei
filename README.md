# Тестовое задание на Младшего Frontend-разработчика в МТЦ Протей

Тестовое задание, реализованное на **React + TypeScript + Vite**.

## Что реализовано

- часть лендинга со списком лекций
- карточки лекций
- форма регистрации на лекцию через React Hook Form
- валидация полей формы
- модальное окно успешной отправки
- модальное окно ошибки
- адаптивная вёрстка

## Технологии

- React
- TypeScript
- Vite
- Tailwind CSS
- React Hook Form

## Локальный запуск проекта

### 1. Клонировать репозиторий

```bash
git clone <ссылка-на-репозиторий>
cd <название-папки-проекта>
```

### 2. Установить зависимости
```bash
npm install
```

### 3. Запустить проект в режиме разработки
```bash
npm run dev
```

После запуска приложение будет доступно в браузере по адресу:
```bash
http://localhost:5173
```

### Структура проекта
```
src/
  components/      # компоненты интерфейса
  data/            # данные лекций
  assets/          # изображения и статические ресурсы
  App.tsx          # корневой компонент
  main.tsx         # точка входа
```

### Особенности реализации
- данные лекций отрисовываются из массива
- выбранная лекция передаётся в форму регистрации
- отправка формы реализована как имитация запроса
- результат отправки показывается через модальные окна
- изображения подключены через import для корректной работы в билде
